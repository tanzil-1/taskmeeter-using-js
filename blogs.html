Question-1: What are the different ways to select an element in the DOM?

In JavaScript, We can select elements using:

  1. document.getElementById("id") - Select by ID.
  2. document.getElementsByClassName("class") - Select by class.
  3. document.getElementsByTagName("tag") - Select by tag name.
  4. document.querySelector("selector") - Select the first matching element.
  5. document.querySelectorAll("selector") - Select all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Returns or sets the HTML content inside an element.
innerText: Returns or sets the visible text inside an element, ignoring hidden content.
textContent: Returns or sets the entire text inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements. This improves performance and makes it easier to manage dynamic elements. It works by using event bubbling to detect clicks on child elements via the parent.
Example: 

document.getElementById("parent").addEventListener("click", function(event) {
    if (event.target.classList.contains("child")) {
        console.log("Child clicked!");
    }
});

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event propagates from the targeted child element up to its parent elements in the DOM tree.
Example: 

document.getElementById("child").addEventListener("click", function() {
    console.log("Child clicked!");
});
document.getElementById("parent").addEventListener("click", function() {
    console.log("Parent clicked!");
});
If you click the child, first **"Child clicked!"** is logged, then **"Parent clicked!"** due to bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements: document.createElement("tag")
Adding Elements: parent.appendChild(element) or parent.insertBefore()
Removing Elements: element.remove() or parent.removeChild(child)

Example:                    
let newDiv = document.createElement("div");
newDiv.textContent = "New Element";
document.body.appendChild(newDiv);

document.body.removeChild(newDiv);
